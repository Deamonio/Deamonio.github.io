GlAISeR

AUTONOMOUS SAFETY ROBOT

GlAISeR는 **Embedded, AI, Web 융합 아키텍처**를 기반으로 작동하는 자율 보안 및 환경 최적화 시스템입니다. 보이지 않는 곳에서 끊임없이 순찰하며 교육 환경의 **안전**과 **효율**을 극대화합니다.

자세한 아키텍처 보기 데모 요청

핵심 목표: 보안과 효율의 자율 융합

🛡️ 자율 보안

YOLO 기반 실시간 객체 인식 및 이상 징후 감지. 인적 개입 없이 위험 요소를 즉시 포착하고 대응하여 안전을 확보합니다.

🔋 에너지 최적화

실내 환경 센서 데이터를 분석하여 냉난방, 조명 시스템을 제어합니다. 자원의 낭비를 제거하고 운영 비용을 절감합니다.

🔗 원격 투명성

모든 운영 데이터와 보안 상황을 웹 대시보드에서 실시간으로 제공하여, 보이지 않는 Daemons의 작업을 투명하게 공유합니다.

GLAISER 융합 아키텍처: 보이지 않는 힘의 구현

🛠️

EMBEDDED LAYER: 기계의 심장부

  • **Daemon Process:** 최소 리소스로 자율 순찰 경로를 관리하는 경량의 OS 코어 프로세스.
  • **Motor Control:** 초저지연 센서 피드백을 통한 정밀하고 부드러운 이동 제어 (SLAM/LiDAR 융합).
  • **Environmental Sensor:** 온도, 습도, CO2 농도 등 환경 데이터를 수집하여 AI 레이어로 전송.
🧠

AI LAYER: 인지 및 예측

  • **Anomaly Prediction:** 순찰 패턴 분석을 통해 잠재적 위험 구역을 미리 예측하고 자율 경로 조정.
  • **Object Recognition:** 저해상도 환경에서도 정확한 사람/사물 인식을 위한 경량화된 YOLOv8 모델 적용.
  • **Optimization Logic:** 에너지 센서 데이터를 기반으로 건물 관리 시스템에 최적의 효율 명령 전송.
🌐

WEB LAYER: 투명한 반영

  • **Live Status Dashboard:** 로봇의 배터리, 위치, 센서 데이터를 실시간으로 시각화.
  • **Event History:** 모든 보안 경고 및 환경 최적화 기록을 불변의 타임라인으로 제공하여 신뢰성 확보.
  • **Remote Override:** 비상 상황 발생 시, 웹 인터페이스를 통한 즉각적인 수동 제어 기능 제공.

기술 사양 및 성능 지표

핵심 성능 지표 (KPI)

자율 탐지 정확도 95% 이상 (YOLOv8-Nano)

평균 시스템 가동 시간 (MTBF) 2,500시간 (무정지 Daemon 설계)

원격 제어 지연 시간 < 100ms (WebRTC 기반)

에너지 절감 효율 월 평균 15% (운영 환경 비교)

AI/Vision

NVIDIA Jetson Nano, PyTorch, YOLOv8-Nano (Optimized for Edge)

Web Interface

React/Next.js, WebRTC for Live Stream, WebSocket for Data Telemetry

Hardware

LIDAR, Depth Camera, Multi-Array Environmental Sensors
